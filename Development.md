# 開発環境を整える

## 手順

1, Androidアプリの開発環境をセットアップする(IDE,Javaなど

2, ソースのダウンロード

    $ mkdir -p ~/Development/android && cd $_
    $ git clone https://github.com/ichigotake/Android-yancha yancha
    $ git submodule init && git submodule update

3, ダウンロードしたソースをIDEのプロジェクトへインポートする

4, 起動してセットアップ完了
